# richmondandcherrie

Wedding website for Richmond and Cherrie.

## GitHub Pages

This site deploys from `main` with GitHub Actions.

Free project URL:

```txt
https://chlyn.github.io/richmondandcherrie/
```

The repository is configured to publish at the URL above.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Lint

```sh
npm run lint
```
